Tree View Component

Instructions:

  • Expand/Collapse: Click the +/− button on nodes with children
  • Add Child: Click the button to add a child node
  • Edit: Double-click on a node label to edit it
  • Delete: Click the × button to remove a node
  • Drag & Drop: Drag nodes to reorder or move them to different parents
  • Lazy Loading: Children are loaded when you first expand a parent node
